# Rapido-Ride-Booking-Analysis(Interactive Dashboard creation using MS Excel)
## Project Objectives
This project is to analyze Rapido ride booking data and create an interactive dashboard in Microsoft Excel that helps monitor booking performance,revenue,ride completion,customer satisfaction, and payment trend.

## Dataset used
-<a href="https://github.com/prajitha-k/Data-Analysis-Dashboard/blob/main/Rapido%20data.xlsx">Dataset</a>

## Questions (KPIs)
1.What percentage of bookings were completed, cancelled, and incomplete?
2.Which payment method (UPI, Cash, Card, or Wallet) is used the most?
3.Which vehicle type (Bike or Auto) receives more bookings?
4.Which city has the highest number of ride pickups?
5.How do customer reviews compare between Good and Bad ratings?
6.What is the distribution of bookings by vehicle type?
7.Which booking status requires operational improvement?
8.Which payment method contributes the highest number of transactions?
9.Which city shows the highest ride demand?
10.What insights can help improve customer satisfaction and ride completion rates?

## process
-Collected the Rapido ride booking dataset.
-Cleaned the data by removing errors and missing values.
-Prepared the data using Excel tables and formulas.
-Created Pivot Tables to summarize the data.
-Designed Pivot Charts to visualize key metrics.
-Added KPIs to monitor business performance.
-Inserted Slicers for interactive filtering.
-Formatted the dashboard with a consistent theme and layout.
-Analyzed booking trends, revenue, payment methods, vehicle types, and ratings.
-Generated actionable insights to support business decision-making.
