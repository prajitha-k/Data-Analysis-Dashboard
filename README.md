# Rapido-Ride-Booking-Analysis(Interactive Dashboard creation using MS Excel)
## Project Objectives
This project is to analyze Rapido ride booking data and create an interactive dashboard in Microsoft Excel that helps monitor booking performance,revenue,ride completion,customer satisfaction, and payment trend.

## Dataset used
-<a href="https://github.com/prajitha-k/Data-Analysis-Dashboard/blob/main/Rapido%20data.xlsx">Dataset</a>

## Questions (KPIs)
-What percentage of bookings were completed, cancelled, and incomplete?
-Which payment method (UPI, Cash, Card, or Wallet) is used the most?
-Which vehicle type (Bike or Auto) receives more bookings?
-Which city has the highest number of ride pickups?
-How do customer reviews compare between Good and Bad ratings?
-What is the distribution of bookings by vehicle type?
-Which booking status requires operational improvement?
-Which payment method contributes the highest number of transactions?
-Which city shows the highest ride demand?
-What insights can help improve customer satisfaction and ride completion rates?
