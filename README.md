# Rapido-Ride-Booking-Analysis(Interactive Dashboard creation using MS Excel)
## Project Objectives
This project is to analyze Rapido ride booking data and create an interactive dashboard in Microsoft Excel that helps monitor booking performance,revenue,ride completion,customer satisfaction, and payment trend.

## Dataset used
<a href="https://github.com/prajitha-k/Data-Analysis-Dashboard/blob/main/Rapido%20data.xlsx">Dataset</a>

## Questions (KPIs)
What percentage of bookings were completed, cancelled, and incomplete?

Which payment method (UPI, Cash, Card, or Wallet) is used the most?

Which vehicle type (Bike or Auto) receives more bookings?

Which city has the highest number of ride pickups?

How do customer reviews compare between Good and Bad ratings?

What is the distribution of bookings by vehicle type?

Which booking status requires operational improvement?

Which payment method contributes the highest number of transactions?

Which city shows the highest ride demand?

What insights can help improve customer satisfaction and ride completion rates?

## process
Collected the Rapido ride booking dataset.

Cleaned the data by removing errors and missing values.

Prepared the data using Excel tables and formulas.

Created Pivot Tables to summarize the data.

Designed Pivot Charts to visualize key metrics.

Added KPIs to monitor business performance.

Inserted Slicers for interactive filtering.

Formatted the dashboard with a consistent theme and layout.

Analyzed booking trends, revenue, payment methods, vehicle types, and ratings.

Generated actionable insights to support business decision-making.

## Project Insights

Completed rides account for the majority of bookings, indicating strong operational performance.

UPI is the most preferred payment method, followed by Wallet and Cash.

Bike rides are booked more frequently than Auto rides.

Delhi records the highest number of ride pickups, showing the greatest demand.

Customer reviews are predominantly positive, with significantly more "Good" reviews than "Bad" reviews.

Average customer rating (4.00) and driver rating (3.99) indicate a high level of user satisfaction.

Cancelled and incomplete rides are relatively low but highlight opportunities to improve ride completion rates.

The interactive dashboard enables quick monitoring of KPIs and supports data-driven decision-making.

## Final Conclusion
The Rapido Ride Analytics Dashboard developed in Microsoft Excel provides a comprehensive overview of ride booking performance through interactive visualizations and KPIs. The analysis reveals that ride completion is high, digital payments (especially UPI) are widely preferred, and customer satisfaction remains strong. This dashboard helps stakeholders identify booking trends, monitor operational efficiency, evaluate customer and driver performance, and make informed business decisions to improve overall service quality and revenue growth.
